HISTOIRE

Technologie web

Le Web est une des applications d’Internet, qui est distincte d’autres applications comme le courrier électronique, la visioconférence et le partage de fichiers en pair à pair. Inventé en 1989-1990 au CERN par Tim Berners-Lee épaulé de Robert Cailliau, le Web a popularisé Internet. Il visait à connecter des données éparpillées sur différents ordinateurs via l'hypertexte, rendant la documentation accessible, unifiée et facile à consulter. Depuis, le Web est fréquemment confondu avec Internet; en particulier, le mot toile est souvent utilisé dans les textes non techniques sans que l'on sache toujours si l'auteur désigne le Web ou Internet.

Langage de Programmation

Un langage de programmation est un langage formel permettant à un être humain d’écrire des instructions compréhensibles par un ordinateur afin qu’il exécute des tâches précises (calculs, gestion de données, création d’applications, etc.). Les langages de programmation ont été créés pour faciliter la communication avec la machine, car au début les ordinateurs étaient programmés en langage machine (suite de 0 et 1), ce qui était extrêmement difficile, long et source d’erreurs. Le but principal était donc de rendre la programmation plus simple, plus lisible, plus rapide et plus efficace, tout en permettant de créer des programmes plus complexes. Le premier langage de programmation considéré comme formel est Plankalkül, conçu dans les années 1940 par Konrad Zuse, bien qu’il n’ait été réellement implémenté que plus tard. L’apparition d’autres langages s’explique par l’évolution des besoins : amélioration de la portabilité entre machines, apparition de nouveaux paradigmes (fonctionnel, orienté objet, logique), développement d’Internet, des applications mobiles, de l’intelligence artificielle et recherche de meilleures performances, sécurité et productivité.

Definitions

Technologie web

Le World Wide Web (/world waɪdˈweb/), abrégé en WWW, W3, le Web, la toile mondiale ou simplement la toile, est un système hypertexte public fonctionnant sur Internet. Le Web permet de consulter, à l'aide d'un navigateur, des pages regroupées en sites. Web signifie littéralement « toile (d’araignée) », image représentant les hyperliens qui lient les pages web entre elles. dans le but principal de faciliter le partage instantané d'informations et de documents entre chercheurs du monde entier.

Langage de programmation

Un langage de programmation est un langage conçu pour exprimer des programmes informatiques . Les langages de programmation permettent généralement d' écrire des logiciels de manière lisible par l'homme . L’exécution d’un programme nécessite une implémentation . Il existe deux approches principales pour implémenter un langage de programmation : la compilation , où les programmes sont compilés en amont en code machine , et l’interprétation , où les programmes sont exécutés directement.

Exemple de Langage de programmation

Technologie web

• JavaScript
    Sert à rendre les pages web interactives (animations, formulaires dynamiques, boutons, mises à jour sans recharger la page). Fonctionne côté navigateur (frontend) et aussi côté serveur avec Node.js.

• PHP
    Utilisé côté serveur pour créer des sites web dynamiques (gestion des bases de données, systèmes de connexion, CMS comme WordPress).

• Python
    Utilisé côté serveur avec des frameworks comme Django ou Flask pour créer des applications web, API et services backend.

• Java
    Utilisé pour développer des applications web robustes et sécurisées côté serveur (Spring Boot, grandes entreprises).

• C#
    Utilisé avec ASP.NET pour développer des sites et applications web performantes.

• TypeScript
    Extension de JavaScript qui ajoute le typage statique pour créer des applications web plus grandes et mieux structurées.

• Ruby
    Utilisé avec Ruby on Rails pour développer rapidement des applications web.

Technologie mobile

• Kotlin
    Utilisé principalement pour développer des applications Android modernes. Il est officiel chez Google pour Android et permet un code plus concis et sécurisé que Java.

• Java
    Ancien langage principal pour le développement Android. Toujours utilisé pour maintenir ou développer certaines applications mobiles.

• Swift
    Langage officiel d’Apple pour créer des applications iOS (iPhone, iPad). Il est rapide, sécurisé et moderne.

• Dart
    Utilisé avec Flutter pour créer des applications mobiles multiplateformes (Android et iOS) avec un seul code source.

• C#
    Utilisé avec .NET MAUI ou Xamarin pour développer des applications mobiles multiplateformes.
